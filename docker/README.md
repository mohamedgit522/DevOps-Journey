# Docker Module - Learning Summary

## Overview
This folder contains projects from the Docker module. The focus was on containerization, Docker fundamentals, and deploying multi-container applications.

---

## Key Topics Covered
- Containers vs. Virtual Machines
- Docker installation and setup
- Writing Dockerfiles & building images
- Containerizing a web application
- Docker networking & linking containers
- Multi-container apps with Docker Compose
- Debugging common container issues
- Docker registries: DockerHub & Amazon ECR
- Image optimization: Multistage builds
- Intro to orchestration: Docker Swarm & Kubernetes

---

## Key Skills Gained
- Building and running Docker containers
- Writing and optimizing Dockerfiles
- Managing multi-container apps with Docker Compose
- Connecting apps to databases (Redis, MySQL) in containers
- Debugging container networking and dependencies
- Pushing/pulling images to/from registries
- Understanding container lifecycle and CLI commands

---

## Highlight Project: Flask + Redis App
- A simple Python Flask web application using Redis for visit counts.
- Implements `/` (welcome) and `/count` (increments page visits) routes.
- Demonstrates multi-container setup using Docker Compose.
- Includes container networking, persistent data volumes, and image management.

---

## Outcome
By completing this module, I gained practical experience in Docker workflows, multi-container apps, and DevOps best practices.
